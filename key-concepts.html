Key Concepts

Six terms that come up constantly in cybersecurity. Understanding these makes the rest of this site easier to follow.

1. The CIA Triad

The three goals of information security: Confidentiality (only authorized people can see data), Integrity (data isn't changed without permission), and Availability (systems are accessible when needed).

2. Least Privilege

A user or program should only have the minimum access it needs to do its job. This limits the damage if an account is compromised.

3. Zero Trust

A security approach that assumes no user or device should be automatically trusted, even inside the network. Every request is verified.

4. Encryption

Converting readable data into a coded form that can only be read again with the correct key.

5. Multi-Factor Authentication (MFA)

Requiring two or more proofs of identity, like a password plus a phone code, before granting access.

6. Patch Management

Regularly applying software updates that fix known security vulnerabilities.
